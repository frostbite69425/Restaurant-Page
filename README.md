# Restaurant-Page

Restaurant Page A restaurant-themed single page website built as part of The Odin Project.
The project focuses on generating and updating all page content dynamically using JavaScript instead of hardcoding HTML. Webpack is used to bundle modules and organize the codebase.

Learning Goals

Create and manage DOM elements using vanilla JavaScript

Dynamically render page content instead of static HTML

Structure a project using JavaScript modules

Use Webpack to bundle and manage assets

Separate logic into reusable components

Practice organizing a scalable front-end codebase

Built as part of The Odin Project to strengthen core JavaScript and frontend architecture skills.
